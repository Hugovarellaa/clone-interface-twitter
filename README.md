## ⚙️ Instalação

```
# Abra um terminal e copie este repositório com o comando
$ git clone https://github.com/Hugovarellaa/clone-interface-twitter
```

```
# Acesse a pasta da aplicação
$ cd clone-interface-twitter

# Instale as dependências
$ yarn

# Inicie a aplicação
$ yarn start

```
